# Vucut Tipi Siniflandirmasi

**Karakter sayisi:** 2,231  

## Yontem Ozeti

| Siniflandirma | Metrik | Guvenirlilk |
|---|---|---|
| Elma / Armut | WHR = waist / hip | Yuksek |
| V-Shape / Dikdortgen | chest / waist orani | Orta-Yuksek |
| Zayif / Normal / Obez | Ponderal Index = height / (volume*1000)^(1/3) | Orta |
| Ektomorf / Endomorf | PI + WHR kombinasyonu | Orta |
